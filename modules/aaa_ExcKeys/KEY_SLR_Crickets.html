Note:
  • When you write the equation of the line don’t use “y”, “x”, “response”, or “explanatory.” Instead make sure you use the name of the actual variables being examined.
  • A statistically significant slope (i.e., statistically different from zero) indicates that there is a statistically significant relationship between the response and explanatory variables. Thus, whether a relationship exists between Y and X depends on if the slope is different from 0 or not.

Results Table – Chirping Crickets

  1. The completed table is shown further below. Note the following.
    • The SE for the intercept is equal to the intercept divided by the t-value for the intercept.
    • The p-value below the coefficient results is exactly the same as the slope p-value.
    • The slope is equal to \(r\frac{s_{temp}}{s_{chirps}}\) where r is the square root of the r2 value.
    • The slope t-value is the slope value divided by the SE of the slope.
    • The slope and intercept values can be used to find the two fitted values (under the predict() results).
    • The lower value in the first predict can be found by finding the difference between the fitted value and the upper value and subtracting this from the fitted value. In other words, the fitted value must be exactly between the lower and upper confidence intervals. A similar thought process can be used to find the upper value of the prediction interval.
    • The mean temperature can be found in two ways. First, note that the intercept is equal to the mean temperature minus the slope times the mean number of chirps. Solve this for mean temperature. Second, the mean temperature can be found by plugging the mean number of chirps into the equation of the line (because the predicted value of y when x is equal to mean x is mean y).
  2. The equation of the best-fit line is Temp = 26.742+3.2163Chirps.
  3. Yes, the p-value for testing that the slope is equal to zero is very small (p=0.0002) leading to a conclusion that the slope is different than zero implying a significant relationship between temperature and the number of chirps.
