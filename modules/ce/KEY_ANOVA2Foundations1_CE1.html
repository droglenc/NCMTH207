Design of a Two-Factor Experiment

Note:
  • Make sure to keep levels separate … need to say the number for each factor.
  • The number of replicates is per treatment … thus, divide total available by the number of treatments.
  • A replicate is an individual or unit of analysis. You should state what that is in this specific case.
  1. The response variable is yield of the product.
  2. The factors are temperature and stirring rates.
  3. There are three levels of temperature and three levels of stirring rates.
  4. There are nine (=3×3) treatments.
  5. There are 4 replicates (=\(\frac{36}{9}\)).
  6. A replicate is a “vessel” in which the product will be heated and stirred.
  7. I would use sample(36) and select the first four numbers for the first treatment, the second four numbers for the second treatment, and so on.

Interaction and Main Effects

Note:
  • Make sure that you comment about each type of effect (interaction, row factor, and column factor) even if that effect is not evident. In other words, explicitly state that there is no effect if there is no effect; don’t just leave that effect absent from your answer.
  • If there is an interaction effect, do NOT say that there is “no main effects.” Rather say that the “main effects cannot be assessed because of the interaction effect.”
  • The “lines” do not have to intersect to represent an interaction, they simply do not “track together.” Imagine “lines” that start near the same point but one goes up and the other goes down. They don’t intersect but they certainly don’t track together.
  • Pay close attention to my descriptions below. Come see me if they do not make sense to you.
  1. There is NO interaction effect because all “lines” “track together.” There is a factor A main effect because all “lines” are not horizontal (implying a difference in factor A level means). There is a factor B main effect because all “lines” are not on top of each other (implying a difference in factor B level means).
  2. There is NO interaction effect because all “lines” “track together.” There is NO factor A main effect because all “lines” are horizontal (implying no difference in factor A level means). There is a factor B main effect because all “lines” are not on top of each other (implying that the factor B level means are not equal).
  3. There IS an interaction effect because the “lines” do not “track together” (in fact they intersect). I will not comment on the main effects because of the presence of the interaction effect.
  4. There is NO interaction effect because all “lines” “track together.” There is a factor A main effect because all “lines” are not horizontal (implying a difference in factor A level means). There is a factor B main effect because all “lines” are not on top of each other (implying a difference in factor B level means).
  5. There is NO interaction effect because all “lines” “track together.” There is a factor A main effect because all “lines” are not horizontal (implying a difference in factor A level means). There is a factor B main effect because all “lines” are not on top of each other (implying a difference in factor B level means).
  6. There IS an interaction effect because the “lines” do not “track together” (in fact they intersect). I will not comment on the main effects because of the presence of the interaction effect.
